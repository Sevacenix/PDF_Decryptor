# GitHub Release Copy

## Repository Description

English:
Desktop utility for batch PDF decryption and batch output naming, with AES support, bilingual UI, and release-ready assets for macOS and Windows.

中文：
一个桌面 PDF 工具，突出批量解密和批量命名输出，支持 AES 解密、中英双语界面，以及可直接下载的 macOS 和 Windows 发布版本。

## Short Feature Intro

English:
Batch PDF Decryptor is a simple desktop utility for users who need to remove PDF passwords without using command-line tools. It can scan files in batches, detect encryption status automatically, decrypt multiple PDFs with one password, and export the results to a chosen folder with flexible batch naming rules.

中文：
Batch PDF Decryptor 是一个面向普通用户的桌面小工具，适合不想使用命令行的场景。它可以批量扫描 PDF、自动检测加密状态、输入一次密码后批量解密多个文件，并按指定规则批量命名输出到目标文件夹。

## Feature Highlights

English:
- Batch import PDF files or entire folders
- Automatic encryption detection
- Batch decryption with a single password
- AES-encrypted PDF support
- Batch output naming rules
- One-click original file name output
- Show or hide password input
- Chinese and English UI support

中文：
- 支持批量导入 PDF 文件或整个文件夹
- 自动检测文件是否加密
- 输入一次密码后批量解密
- 支持 AES 加密 PDF

- 支持中英文界面

English:
This release delivers the polished public desktop build of Batch PDF Decryptor. It focuses on the full batch workflow: import encrypted PDFs in bulk, detect protection status automatically, remove passwords with one shared password, and export clean files with consistent output naming.

中文：
