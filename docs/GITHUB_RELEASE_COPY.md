# GitHub Release Copy

## Repository Description

English:
Lightweight desktop PDF decryptor with batch processing, AES support, and a bilingual Chinese-English interface.

中文：
一个轻量级桌面 PDF 解密工具，支持批量处理、AES 解密，以及中英双语界面。

## Short Feature Intro

English:
PDF Decryptor is a simple desktop utility for users who need to remove PDF passwords without using command-line tools. It can scan files in batches, detect encryption status automatically, decrypt multiple PDFs with one password, and export the results to a chosen folder with flexible naming rules.

中文：
PDF Decryptor 是一个面向普通用户的桌面小工具，适合不想使用命令行的场景。它可以批量扫描 PDF、自动检测加密状态、输入一次密码后批量解密多个文件，并按指定规则输出到目标文件夹。

## Feature Highlights

English:
- Batch import PDF files or entire folders
- Automatic encryption detection
- Batch decryption with a single password
- AES-encrypted PDF support
- Custom output naming rules
- One-click original file name output
- Show or hide password input
- Chinese and English UI support

中文：
- 支持批量导入 PDF 文件或整个文件夹
- 自动检测文件是否加密
- 输入一次密码后批量解密
- 支持 AES 加密 PDF
- 支持自定义输出命名规则
- 一键使用原文件名输出
- 支持显示或隐藏密码
- 支持中英文界面
