# GitHub Publish Checklist

## Before First Push

- Review `.gitignore` and make sure `build/`, `dist/`, and cache files are not committed by mistake
- Confirm that `LICENSE` matches the license you want to publish with
- Confirm the app name, repository name, and short description
- Add at least one screenshot if you want a stronger project page

## Recommended Repository Structure

- `README.md`: English landing page
- `README.zh-CN.md`: Chinese documentation
- `docs/GITHUB_RELEASE_COPY.md`: repo description and release copy
- `scripts/build_macos_app.sh`: macOS packaging script

## Recommended Release Asset

- `dist/PDF_Decryptor-macOS.zip`
- `dist/PDF_Decryptor-windows.zip`
